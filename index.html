IT And Network Administrator at UDC-Studios

October 2021 to Present

  • UDC-Studios is a non-profit organization that provides remote services for the community of 'Traiskirchen, Österreich', and select members around the world. Remote services can include storage, media, and dedicated servers for game hosting. The main games that UDC-Studios host are Satisfactory, Minecraft, etc. UDC-Studios also provides workstations for users who don't have high-end computers and need to do work quickly. The workstations are equipped with powerful graphics cards and processors to handle demanding tasks efficiently.

    My job in this organisation was to ensure a secure connected between the server, to the user. The first interaction between UDC-Studios and the client will be through VPN tunnel. I configured the VPN to allow only whitelisted IP addresses into the network. The VPN is using IKEv2 AES-256, as it provides a secure key whilst still being quick. Each client will be equipped with a domain user via a local Microsoft Active Directory. Having domain users allows each client to authenticate to their own users on all devices connected to the domain. Domain users are also much easier to configure and administrate, because user data is stored on a centralised server.

    Because this is a public service, there will still be many people who attempt to breach the servers. I have designed a PowerShell Script which gathers details about the events stored on the server, (specifically the user authentication events). The script will cache the data and send it to the local NAS to be analysed more clearly, (Windows event manager isn't the most user-friendly interface). Another script is used to send a message to the Administration team that there have been numerous failure sign-in attempts.
